## Getting Started

**MailMergeKit** makes it easy to create personalized draft emails in Outlook using Word's mail merge data.

### Prerequisites

- Windows 10 or later
- Microsoft Word 2016+ (desktop version)
- Microsoft Outlook 2016+ (desktop version)
- .NET Framework 4.8+

---

## Installation

[Coming in v0.1.0]

1. Download `MailMergeKit-Setup-v0.0.1.msi`
2. Run as administrator
3. Restart Word
4. Find **MailMergeKit** button in Mailings tab

## FAQ

**Q: Does this work with Outlook.com (web)?**  
A: No, desktop Outlook only.

**Q: How many emails can I send?**  
A: Up to ~5,000 emails (Outlook limitation).

**Q: Is my data sent to the cloud?**  
A: No, 100% local processing.
